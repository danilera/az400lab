# POC

## App

### Dependências

* Node Js 12+
* NPM 6+

### Dependências de desenvolvimento

* Webpack
* Webpack dev server
* Sass
* Babel

### Para instalar as dependências

```javascript
cd app && yarn install
```

### Para subir o app

```javascript
cd app && yarn serve
```

Irá subir um servidor na porta 3000

### Para compilar para produção

```javascript
yarn build
```
